# MiniOS v2.1.2 

## Overview
This update brings many UI/UX improvoments along with firmware stability.

## New Features
- Terminal cursor indicator – improved visibility while typing
- Ping utility enhancement – option to specify the number of pings
- Enhanced themes – more colourful prompts for a better visual experience


## Improvements
- **Improved Ping Pong game** - the ball now changes colour every 5 points (up to 20). Try to spot them all!
- **Display buffering improvement** - out of bound scrolling issues are fixed.
- **S3 devkit pin config improvement** 
``` c
    #define TFT_MOSI 17
    #define TFT_SCLK 18
    #define TFT_CS 14
    #define TFT_DC 15
    #define TFT_RST  4 
```
